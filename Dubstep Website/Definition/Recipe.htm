Dubstep Recipe

Ingredients

  • Electricity
  • Button Pad
  • Mixing Software
  • Quick Reflexes

Process

  1. Start with a melody
  2. Add some funk to the instrumental line
  3. Tear it apart
    • Push ahead
    • Pull back
  4. Add a drop
  5. Finish with all elements in effect
  6. Last few seconds should be soft like before a drop
